# Assets

Drop these image files into this folder. The site references them by these exact names:

| File | What it is | Suggested size |
|------|------------|----------------|
| `logo.png` | The caidal app icon shown in the header's white tile. | ~512×512 px, square, transparent or white background |
| `favicon.png` | The little icon in the browser tab. | 32×32 or 64×64 px |
| `og-image.png` | The preview image shown when the site is shared on social/messages. | 1200×630 px |

Notes:
- **`logo.png` is the only one needed for the header to look right.** The other two are nice-to-haves.
- PNG is fine; SVG also works for the logo if you'd rather (update the `<img src>` in the HTML to `assets/logo.svg`).
- You can export `logo.png` straight from your app's icon asset.
